# People Counting

*Bisa AI Webinar*

## **Requirements**
1. Opencv 3.4.2
2. Imutils 0.5.3
3. Tensorflow 2.3.0

## **How to**
1. Clone this repository
2. Download a TF 2 Object Detection model from TF Model zoo
3. put the model inside people-counting repository
4. Change model and video filename in `detection_video.py` script.
5. Run!
